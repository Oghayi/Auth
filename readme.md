#Work Tracker API
A secure and scalable backend API for tracking daily work, tasks and reminders.

Built with Node.js, Express, MongoDB and JWT authentication. This project demonstrates real word backend architecture including authentication, authorization, scheduled jobs.

## Features

1. User Authentication
2. JWT based authorization
3. Role based access (User/Admin)
4. Create, update and delete tasks
5. Reminder system
6. Automated cron jobs for due tasks
7. RESTful API design
